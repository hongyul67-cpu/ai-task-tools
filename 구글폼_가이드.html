📝 구글 폼으로 제출받기 — 설정 가이드

학생이 파일을 직접 보내는 대신 익숙한 구글 폼에 올리게 하는 방법입니다. 한 번 만들어 두면 제출 파일이 선생님 드라이브에 학생별로 자동 정리되고, 폼 응답 시트에서 제출 현황을 한눈에 볼 수 있습니다.

① 구글 폼 만들기 (약 3분)

  1. 구글 폼 접속forms.google.com에 학교(또는 개인) 구글 계정으로 로그인 → 새 양식(＋) 클릭.
  2. 폼 제목 정하기예: 실습Ⅱ-1-01 과제 제출 — 과제 이름과 같게 하면 관리가 편합니다. 학기 내내 하나의 폼으로 모든 과제를 받으려면 ○○과목 과제 제출함처럼 만들어도 됩니다.
  3. "파일 업로드" 문항 추가문항 추가(＋) → 문항 유형을 파일 업로드로 변경 → "계속" 확인. 문항 제목은 과제제출 파일(.html)을 올려주세요로 적으세요.
  4. 파일 업로드 세부 설정· 특정 파일 형식만 허용: 끔(또는 켠다면 유형에 제한 없음)
    · 최대 파일 수: 1   · 최대 파일 크기: 10MB (사진 많은 과제면 100MB)
    · 문항을 필수로 설정 ✔
  5. (추천) 학반·번호·이름 문항 추가단답형 문항으로 학반, 번호, 이름을 추가해두면, 파일을 열지 않아도 폼 응답만으로 누가 냈는지 바로 확인됩니다.
📋 파일 업로드 문항에 이렇게 안내 문구를 적어두면 좋아요:
"제출 도구에서 [제출(구글 폼)] 버튼을 누르면 과제제출.html 파일이 저장됩니다.
그 파일을 여기에 올려주세요. 보통 다운로드 폴더에 있습니다. 파일은 1개만!"

② 과제 양식에 폼 연결하기 (약 1분)

  1. 폼 링크 복사폼 오른쪽 위 보내기 → 🔗(링크) 탭 → "URL 단축" 체크 → 복사.
  2. 양식 만들기 도구에 붙여넣기양식 만들기 도구의 "제출받을 구글 폼 링크" 칸에 붙여넣고 양식을 저장하세요.
  3. 끝!이 양식으로 과제를 하는 학생은 제출 버튼이 "제출(구글 폼)"으로 표시되고, 누르면 파일 저장 → 폼이 자동으로 열립니다.
✅ 폼 링크를 넣지 않은 양식은 기존처럼 공유·파일 전달 방식으로 제출됩니다. 과제마다 골라 쓸 수 있어요.

③ 제출 확인하고 채점하기

  1. 제출 현황 확인폼의 응답 탭에서 누가 제출했는지 확인. 응답을 시트로 연결해두면 표로 볼 수 있습니다.
  2. 제출 파일 받기파일은 내 드라이브의 폼 이름 폴더 → "파일 응답" 폴더에 자동으로 모입니다. 파일명 뒤에 제출자 계정 이름이 자동으로 붙습니다.
  3. 채점드라이브에서 파일을 내려받아 채점 도구로 열면 됩니다. 여러 명을 한 번에 받으려면 폴더에서 전체 선택 → 다운로드(zip) → 풀어서 채점 도구의 "CSV로 내보내기"에 한꺼번에 선택할 수도 있습니다.

④ 자주 겪는 문제

학생이 폼에 접속하면 "권한이 필요합니다"라고 나와요.
파일 업로드 문항이 있는 폼은 구글 로그인이 필요합니다. 학교 계정(Workspace) 폼이라면 설정 ⚙ → 응답 → "조직 외 사용자 응답 허용" 여부를 확인하세요. 학생이 학교 계정이 있다면 그 계정으로 로그인하게 하는 것이 가장 깔끔합니다.
학생이 어떤 파일을 올려야 하는지 헷갈려해요.
제출 도구가 저장하는 파일은 과제제목_학번_이름.과제제출.html 형식입니다. 폼 안내 문구에 "다운로드 폴더에서 .과제제출.html로 끝나는 파일"이라고 적어주세요. 제출 완료 화면에도 같은 안내가 자동으로 표시됩니다.
학생이 폼 제출을 잊어요.
제출 도구의 완료 화면이 "① 폼 열기 → ② 파일 선택 → ③ 제출" 3단계를 안내하지만, 수업 중 "폼 제출까지 눌러야 제출 완료"라고 한 번 말해주시면 좋습니다. 폼 응답 탭과 출석부를 대조하면 누락자를 바로 찾을 수 있습니다.
같은 학생이 두 번 제출했어요.
드라이브 파일명과 응답 시트의 타임스탬프로 최신 것을 확인하고 그 파일로 채점하면 됩니다. 설정 ⚙에서 "응답 1회로 제한"을 켤 수도 있으나, 수정 제출이 막히므로 권장하지 않습니다.
휴대폰으로 제출하는 학생은요?
동일하게 동작합니다. 제출 버튼 → 파일이 휴대폰에 저장 → 폼이 열리면 파일 선택에서 "파일" 앱(안드로이드는 다운로드, 아이폰은 파일 앱)에서 방금 저장된 파일을 고르면 됩니다.
