⚡ 자동 업로드 설정 가이드 (Apps Script)

학생이 제출 버튼만 누르면 제출 파일이 선생님 드라이브로 자동 업로드되는, 가장 편한 제출 방식입니다. 구글 계정만 있으면 되고, 설정은 한 번(약 10분)이면 끝납니다.

① 스크립트 만들기 (약 5분)

  1. Apps Script 열기script.google.com에 구글 계정으로 로그인 → 새 프로젝트 클릭.
  2. 프로젝트 이름 정하기왼쪽 위 "제목 없는 프로젝트"를 눌러 과제 자동제출함 등으로 변경.
  3. 아래 코드 전체를 복사해 붙여넣기편집기에 원래 있던 function myFunction()...은 지우고, 아래 코드로 전부 바꿉니다.
// 과제 자동제출함: 학생 제출 파일을 드라이브에 저장합니다.
// 저장 위치: 내 드라이브 / 과제 제출함 / (과목) / (과제명) / 파일
var ROOT_FOLDER = '과제 제출함';   // 원하는 최상위 폴더 이름으로 바꿔도 됩니다

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    if (!data || !data.content) throw '내용이 비어 있습니다';
    var folder = getFolder_([ROOT_FOLDER, data.subject, data.assignment]);
    var name = String(data.filename || '제출.html');
    // 같은 학생이 다시 제출하면 시각을 붙여 보존
    var stamp = Utilities.formatDate(new Date(), 'Asia/Seoul', 'MMdd_HHmm');
    if (folder.getFilesByName(name).hasNext()) {
      name = name.replace(/\.html$/i, '') + '_' + stamp + '.html';
    }
    folder.createFile(Utilities.newBlob(data.content, 'text/html', name));
    return out_({ ok: true });
  } catch (err) {
    return out_({ ok: false, error: String(err) });
  }
}

function getFolder_(names) {
  var f = DriveApp.getRootFolder();
  for (var i = 0; i < names.length; i++) {
    var n = String(names[i] || '').trim();
    if (!n) continue;
    var it = f.getFoldersByName(n);
    f = it.hasNext() ? it.next() : f.createFolder(n);
  }
  return f;
}

function out_(o) {
  return ContentService.createTextOutput(JSON.stringify(o))
    .setMimeType(ContentService.MimeType.JSON);
}

② 웹 앱으로 배포하기 (약 3분)

  1. 배포 시작오른쪽 위 배포 → 새 배포 → ⚙(유형 선택) → 웹 앱.
  2. 설정 두 가지 (중요!)· 다음 사용자 인증 정보로 실행: 나
    · 액세스 권한이 있는 사용자: 모든 사용자 ← 이걸로 해야 학생이 로그인 없이 제출됩니다.
  3. 권한 승인배포 클릭 → "액세스 승인" → 내 계정 선택 → "고급" → "…(안전하지 않음)으로 이동" → 허용. (내가 만든 스크립트라 뜨는 표준 경고입니다.)
  4. 웹 앱 URL 복사https://script.google.com/macros/s/…/exec 형태의 URL을 복사합니다.
⚠️ 코드를 나중에 수정하면 배포 → 배포 관리 → ✎(수정) → 새 버전으로 다시 배포해야 반영됩니다. URL은 그대로 유지됩니다.

③ 과제 양식에 연결하기 (약 1분)

  1. 양식 만들기 도구 열기과제 양식 만들기 도구의 "자동 업로드 주소 (Apps Script)" 칸에 복사한 URL을 붙여넣습니다.
  2. 양식 저장 → 배포이 양식으로 과제를 하는 학생은 제출 버튼이 "제출하기"가 되고, 누르면 자동 업로드 후 "제출 완료!" 화면이 나옵니다.
  3. 제출 확인내 드라이브의 과제 제출함 → 과목 → 과제명 폴더에 파일이 실시간으로 쌓입니다. 같은 학생이 다시 내면 시각이 붙어 저장됩니다.
✅ 한 번 배포한 URL은 모든 과제에 재사용할 수 있습니다. 과목·과제명 폴더가 자동으로 나뉘므로 과제마다 새로 만들 필요가 없습니다.
💡 자동 업로드가 실패하면(인터넷 문제 등) 도구가 자동으로 파일 저장 방식으로 전환해 학생이 직접 제출할 수 있게 안내합니다. 구글 폼 링크도 함께 넣어두면 폴백 시 폼 안내가 나옵니다.

④ 자주 묻는 질문·보안

학생이 구글 로그인을 해야 하나요?
아니요. "모든 사용자" 액세스로 배포하면 로그인 없이 제출됩니다. (구글 폼 방식과의 가장 큰 차이입니다.)
URL을 아는 사람이 내 드라이브를 볼 수 있나요?
아니요. 이 스크립트는 파일을 새로 만드는 것만 할 수 있고, 드라이브를 읽거나 다른 파일을 보는 기능이 없습니다. URL이 퍼지면 장난 제출이 올 수는 있으니 학급 안에서만 공유하세요(어차피 양식 파일 안에만 들어 있습니다).
업로드 용량 제한이 있나요?
한 번에 수십 MB까지 처리됩니다. 사진이 아주 많은 과제도 문제없는 수준입니다.
제출이 됐는지 학생이 어떻게 아나요?
업로드가 실제로 성공해야 "제출 완료!" 화면이 나옵니다. 실패하면 파일 저장 방식으로 자동 전환되므로, "제출 완료!"를 봤다면 드라이브에 들어간 것입니다. 원하면 학생이 "내 기기에도 사본 저장"을 눌러 증빙을 남길 수 있습니다.
학교 Workspace 계정에서 "모든 사용자" 옵션이 없어요.
관리 정책으로 외부 공개 배포가 막힌 경우입니다. 개인 구글 계정으로 스크립트를 만들거나, 관리자에게 문의하거나, 구글 폼 방식을 사용하세요.
