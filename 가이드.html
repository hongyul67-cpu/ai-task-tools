🔄 전체 흐름 한눈에 보기

세 도구는 서로 자동 연결되지 않습니다. JSON 파일을 직접 주고받으며 사용합니다.

교사: 양식 만들기→ 양식.json 배포→ 학생: 작성·제출
제출 파일(.html) 1개 전달→ 교사: 채점→ 피드백 PDF 전달
📌 파일 전달은 카카오톡, 이메일, 구글 클래스룸, USB 등 평소 쓰던 방법을 그대로 사용하면 됩니다. 도구가 자동으로 보내주지 않습니다.
📋 과제 목록은 여러 과목을 담을 수 있습니다. 상단에서 과목을 전환하고, "과제 추가"로 과제를 넣은 뒤 "목록 저장"으로 과목 목록 파일(.json)을 내보낼 수 있습니다. 다른 기기에서는 "불러오기"로 그 파일을 열면 됩니다. 학생에게는 그 파일을 저장소에 올린 뒤 "학생용 링크"(과제목록.html?list=경로)를 만들어 과목별로 나눠주면, 학생은 자기 과목 링크만 열면 됩니다. 처음 보이는 AI 과목은 예시이며, 필요 없으면 과목 삭제로 지울 수 있습니다.

교사용 ① 과제 양식 만들기

학생에게 배포할 과제 양식(.json)을 만드는 도구입니다.

  1. 기본 정보 입력과제 제목(필수), 과목명, 담당 교사, 과제 설명을 적습니다.
  2. 질문(문항) 만들기상단 템플릿(캡처 제출형 / 글 작성형 / 링크 제출형 / 글+사진 혼합형 / 전체 혼합형)을 누르면 자동으로 질문이 채워집니다. "질문 추가하기"로 직접 추가도 가능합니다.
  3. 문항별 옵션 설정각 질문마다 ✏️ 글 입력 / 📷 사진 첨부 / 🔗 링크 중 학생이 쓸 수 있는 항목을 체크합니다. 여러 개 동시 선택 가능합니다.
  4. 미리보기로 확인하단 "미리보기" 버튼으로 학생이 보게 될 화면을 점검합니다.
  5. 양식 파일 저장"양식 파일 저장"을 누르면 과제양식_제목.json 파일이 다운로드됩니다.
  6. 학생에게 배포이 .json 파일을 학급 게시판, 메신저, 클래스룸 등으로 학생에게 전달합니다.
💡 이미 만든 양식을 수정하려면 "③ 기존 양식 불러와서 수정"에서 그 .json 파일을 다시 열면 됩니다.
→ 양식 만들기 도구 열기

학생용 ② 과제 작성·제출

선생님이 준 양식(.json)을 열어 답을 작성하고, 제출 파일 하나(.html)로 제출하는 도구입니다.

  1. 과제 양식 열기"과제 양식 파일 열기"로 선생님께 받은 .json 파일을 선택합니다.
  2. 제출자 정보 입력학반·학번·이름을 입력합니다 (이름은 필수).
  3. 문항별로 답 작성 ✏️ 글 입력: 텍스트 칸에 답을 입력
    📷 사진 첨부: "카메라로 촬영" 또는 "앨범에서 선택"으로 이미지 첨부 (여러 장 가능)
    🔗 링크 첨부: 구글드라이브/유튜브 등의 링크를 붙여넣기 (공유 설정을 "링크가 있는 모든 사용자"로 바꿔야 선생님이 열 수 있음)
  4. 자동저장 확인30초마다 자동으로 저장되며, 브라우저를 꺼도 같은 기기·브라우저에서는 이어서 작성할 수 있습니다.
  5. 미리보기로 점검"미리보기" 버튼으로 빠진 항목이 없는지 확인합니다.
  6. 제출(공유)"제출(공유)"을 누르면 과제제출.html 파일 하나가 만들어집니다. 이 파일 안에 답안과 채점용 정보가 모두 담겨 있습니다. 모바일에서는 공유 시트가 열려 카카오톡·메일 등으로 바로 보낼 수 있습니다.
  7. 선생님께 전달이 파일 하나만 선생님께 보내면 채점이 가능합니다. (원하면 미리보기의 "PDF 저장(선택)"으로 인쇄·보관용 PDF도 따로 만들 수 있습니다.)
📄 제출 파일(.html)은 더블클릭하면 학생 본인 답안을 그대로 볼 수 있고, 선생님 채점도구에서 열면 데이터가 자동 인식됩니다. PDF 라이브러리가 필요 없어 인터넷이 불안정해도 제출됩니다.
💾 "파일저장" 버튼으로 작업 중인 내용을 .json으로 저장해두면, 다른 기기에서도 "작성하던 과제 이어서 하기"로 이어서 작성할 수 있습니다.

교사용 ③ 채점·피드백

학생이 보낸 제출파일(.html)을 열어 채점하고 피드백 PDF를 만드는 도구입니다.

  1. 제출파일 열기"과제제출 파일 열기"로 학생에게 받은 .과제제출.html 파일을 선택합니다. (예전 .json 제출 파일도 그대로 열립니다.)
  2. 답안 확인학생이 작성한 글, 첨부한 사진(클릭하면 확대), 링크(클릭하면 새 탭에서 열림)를 확인합니다.
  3. 문항별 피드백 작성각 문항 아래 피드백 칸에 의견을 적습니다. 자주 쓰는 문구는 "⚡ 자주 쓰는 피드백" 영역에 등록해두면 클릭 한 번으로 삽입할 수 있습니다.
  4. 전체 총평 작성 (선택)맨 아래 "전체 총평" 칸에 종합 의견을 남길 수 있습니다.
  5. 피드백 PDF 저장"피드백 PDF" 버튼을 누르면 학생 답안 + 피드백이 합쳐진 PDF가 다운로드됩니다.
  6. 학생에게 전달완성된 PDF를 학생에게 다시 보내줍니다.
📊 화면 상단의 "채점 현황" 대시보드는 같은 브라우저에서 채점한 학생 목록을 자동으로 기록해 보여줍니다 (브라우저 기기 안에만 저장됨).
💾 채점을 다 못 끝냈다면 "채점저장"으로 진행 중인 내용을 .json으로 저장해두고, 나중에 다시 열어 이어서 채점할 수 있습니다.

❓ 자주 묻는 질문

인터넷이 없어도 쓸 수 있나요?
페이지를 한 번 열면 그 다음부터는 인터넷 연결 없이도 작동합니다(단, 처음 접속할 때는 필요).
학생 답변이 외부 서버로 전송되나요?
아니요. 모든 데이터는 학생의 브라우저 안에서만 처리되고, 사람이 직접 파일로 내보내 전달하는 방식입니다. 자동으로 어디에도 업로드되지 않습니다.
브라우저를 바꾸거나 기기를 바꾸면 작성한 내용이 사라지나요?
자동저장은 같은 브라우저·같은 기기에만 남습니다. 다른 기기로 이어서 작성하려면 "파일저장"으로 작업파일(.json)을 받아 새 기기에서 "작성하던 과제 이어서 하기"로 불러오세요.
제출할 때 어떤 파일을 보내야 하나요?
.과제제출.html 파일 하나만 보내면 됩니다. 이 파일 하나에 답안(보기용)과 채점용 데이터가 모두 들어 있어, 선생님이 채점도구에서 바로 열 수 있습니다. PDF는 필요할 때만 따로 저장하는 선택 사항입니다.
링크를 첨부했는데 선생님이 못 열어요.
구글드라이브는 공유 설정을 "링크가 있는 모든 사용자"로, 유튜브는 공개 범위를 "공개" 또는 "일부 공개"로 바꿔야 합니다. 비공개 상태면 선생님이 열 수 없습니다.
